Data & Storage
Scripts
All scripts are saved to this browser's localStorage under the key wrotenote_scripts. Data lives in this browser only — it does not sync across devices or browsers.
Exports
Exported files are downloaded to your browser's default Downloads folder:
• .wrotenote — full script JSON (Scripts panel)
• .txt — plain text export (Scripts panel)
• .wn-breakdown — breakdown session JSON (Breakdowns panel)
• .ttx / .pdf — Final Draft XML & browser print PDF (Breakdowns panel)
Preferences
UI settings (theme, density, page numbers, page splitters) are saved to localStorage under keys prefixed wn_.
